## 🧠 What is this?

Este repositorio explora una idea peligrosa:

> ¿Y si elegir una distro Linux no fuera una decisión lógica…  
> sino una extensión de tu personalidad?

Para responder eso, hicimos lo obvio:

👉 Aplicar Machine Learning al comportamiento de alpacas.

## 🧪 The Experiment

Simulamos datos de comportamiento alpaca con features como:

- 🌿 Grazing Variance (¿comes tranquilo o con ansiedad?)
- 🦙 Social Dominance (¿lideras… o sigues?)
- 🚧 Fence Interaction (¿respetas límites?)
- ⚡ Stress Response (¿te rompes bajo presión?)

Luego aplicamos:

- Clustering con K-Means  
- Reducción dimensional con PCA  
- Embeddings con Autoencoders 👀  

Y dejamos que los datos hablen.

Spoiler: hablaron demasiado.

## 🧬 The Result

Los clusters emergentes no eran aleatorios.

Eran… personalidades.

Y cada personalidad se alineaba inquietantemente bien con filosofías de Linux:

---

### 🦙 The Control Freak  
→ Arch Linux  

- No confía en nadie  
- Construye todo desde cero  
- Si algo falla… lo rompiste tú  

> “Esta alpaca no quiere comida. Quiere control del pipeline.”

---

### 🦙 The Traditionalist  
→ Debian  

- Rutinas claras  
- Estabilidad absoluta  
- Cero drama  

> “Algunos sistemas duran décadas… porque fueron diseñados sin prisa.”

---

### 🦙 The Corporate Mind  
→ Red Hat Enterprise Linux  

- Estructura  
- Soporte  
- Responsabilidad  

> “No corre. Factura.”

---

### 🦙 The Explorer  
→ Fedora  

- Prueba todo  
- Rompe cosas  
- Vive en el borde  

---

### 🦙 The Peaceful One  
→ Ubuntu

> Simplicidad no es debilidad. Es diseño.
